# ServerSmith

A Java 21 + JavaFX application for managing Minecraft servers.

## Prerequisites
- Java 21 (JDK)
- Gradle (Wrapper included)

## How to Run
```bash
./gradlew run
```

## How to Build
```bash
./gradlew build
```

## Project Structure
The source code is located in `src/main/java/dev/perillo/serversmith`.
Configuration is in `build.gradle.kts`.
